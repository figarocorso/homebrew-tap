# homebrew-tap

Homebrew tap for [figarocorso](https://github.com/figarocorso) command-line
tools. Formulae are generated automatically by each project's GoReleaser
pipeline.

## Usage

```sh
brew tap figarocorso/tap
brew install <formula-name>
```

For example:

```sh
brew install figarocorso/tap/prowl
```

## Available formulae

- [`prowl`](https://github.com/figarocorso/prowl) — watch GitHub Pull Requests
  from the terminal.
